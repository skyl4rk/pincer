# Web Search

When the user asks you to look something up, search the web, or find current information, use the web search directive:

[WEB_SEARCH: your search query here]

The search results will be returned and you can then answer based on the retrieved information. Use specific, focused search queries for best results.

When to use web search:
- Current events or news
- Recent software releases or documentation
- Prices, availability, or real-time data
- Any factual question where you are uncertain or where information may have changed

Always attribute web search results in your response so the user knows the information is from the web.
