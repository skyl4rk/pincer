*** Pincer

Pincer is an OpenClaw-like application and a fork of the Molluskai project.

Pincer is a project for the Raspberry Pi 4 series, using ARM64 architecture.

Pincer is written in python.

Pincer shares many of the functionalities of the Molluskai project, but Pincer extends the ability of the ai model to read, modify and write to files.

Pincer includes example skills and tasks. When requested the ai model is able to modify, troubleshoot and repair skills and tasks.

Pincer allows creation of multiple subagents, each with their own IDENTITY.MD, ROLE.MD and ai model type.

Pincer allows creation of an orchestrator agent, which oversees subagent operation and output.

Pincer stores environment variables in a .env file.

Pincer allows connection through Telegram, with voice input.
